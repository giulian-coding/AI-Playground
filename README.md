# AI-Playground

A personal playground for experimenting and learning in the field of artificial intelligence and machine learning.

## About this Project

This repository serves as a learning environment to explore various AI and ML concepts hands-on. Different approaches and technologies are explored here - from implementing neural networks from scratch to using modern frameworks.

## Content

- **AI from Scratch** - Neural networks implemented from scratch with NumPy
- **AI with Keras** - Experiments with TensorFlow/Keras
- **Knowledge Base** - Collected notes and concepts

## Technologies

- Python
- NumPy
- TensorFlow / Keras
- Jupyter Notebooks
